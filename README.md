# 微博相册图片获取工具
Author：李小恩
## 使用

打开`main.py`根据注释配置好对应信息即可
以下是示例
```python
# 新浪用户的uid
uid = 1947117325
# 相册id
album_id_list = [
    189229,
    3558779514364152,
]
# 你的新浪账号cookie
cookie = 'cookie'
# 链接格式可用变量{pic_id}
pic_format = 'https://tva1.sinaimg.cn/large/{pic_id}.jpg'
```
